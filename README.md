# My Learning Journey

A small Next.js site that documents a simple learning journey in web development.

## Overview

This project is a personal space for sharing progress, goals, and short notes on frontend, backend, and mobile development.

## Pages

- `Home` — a brief summary of the learning journey.
- `About` — a short introduction with a quick snapshot of skills and focus.
- `Blog` — a minimal blog section with frontend, backend, and mobile categories.
- `Contact` — a simple contact page with a single email address.

## Start the Project

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Edit the Site

Update page content in `src/app`:

- `src/app/page.tsx`
- `src/app/about/page.tsx`
- `src/app/blog/page.tsx`
- `src/app/contact/page.tsx`

## Notes

This project uses Next.js App Router and a lightweight page structure for easy edits and fast iteration.
